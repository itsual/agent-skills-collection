# Secondary Data Analysis

## Overview

Secondary analysis extracts value from data already collected. It is fast when fit is good, and dangerous when the data’s original purpose mismatches the new question.

## When to Use

- Product analytics and log analysis for research questions
- Public statistics and open datasets
- Re-analysis of prior survey or study data
- Triangulation alongside primary research

## Core Practices

- Understand original collection purpose, definitions, and biases
- Assess coverage, missingness, and measurement quality
- Align units and time windows to the research question
- Document transformations and exclusion rules
- Avoid overclaiming causality from observational data
- Combine with primary methods when gaps are material

## Principles

- Convenience data is not automatically representative
- Proxy metrics need explicit justification
- Reproducible pipelines beat one-off spreadsheet archaeology
- Privacy and permitted-use constraints still apply

## Verification

- [ ] Data provenance and definitions are documented
- [ ] Limitations relative to the question are stated
- [ ] Analysis steps are reproducible
