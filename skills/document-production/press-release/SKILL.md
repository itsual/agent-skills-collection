# Press Release

## Overview

A press release packages news so it can be understood and reused quickly. It should be factual, scannable, and written in inverted-pyramid style.

## When to Use

- Product or feature launches
- Funding or acquisition announcements
- Partnerships and major deals
- Executive appointments or organizational news
- Event announcements

## Standard Structure

1. Headline (and optional subheadline)
2. Dateline + opening paragraph (who, what, when, where, why)
3. Supporting details and context
4. Quotes from relevant spokespeople
5. Boilerplate about the organization
6. Media contact information

## Principles

- Put the news in the first paragraph
- Use concrete facts over hype
- Keep quotes substantive, not empty praise
- Write in third person
- Make it easy to copy and reuse

## Verification

- [ ] The core news is clear in the first 2–3 sentences
- [ ] Essential facts (who/what/when/where) are present
- [ ] Contact information is included
- [ ] Tone is professional and credible
