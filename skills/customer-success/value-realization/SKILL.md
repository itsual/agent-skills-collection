# Value Realization

## Overview

Value realization makes outcomes visible. Customers renew more confidently when they can see and tell the story of impact inside their organization.

## When to Use

- Defining success metrics at onboarding
- ROI and value review conversations
- Executive sponsorship reinforcement
- Renewal and expansion justification

## Core Practices

- Establish baseline metrics before or early in deployment
- Align value metrics to the customer’s KPIs, not only vendor vanity metrics
- Collect quantitative and qualitative proof points
- Package value stories for internal customer stakeholders
- Update value evidence ahead of renewal cycles
- Share anonymized patterns back into sales and product

## Principles

- Value unmeasured is value forgotten at renewal time
- Customer-owned metrics beat vendor-imposed dashboards when stakes are high
- Correlation claims should be honest about attribution limits
- Champions need ammunition for their internal buyers

## Verification

- [ ] Baseline and target metrics are set for strategic accounts
- [ ] Value evidence is refreshed on a cadence
- [ ] Renewal conversations reference realized outcomes
