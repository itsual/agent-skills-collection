# Onboarding Success

## Overview

Onboarding is where retention is often won or lost. Strong onboarding reaches first value quickly, sets expectations, and establishes a success baseline.

## When to Use

- New-customer implementation programs
- Improving time-to-first-value
- Reducing early churn
- Standardizing onboarding playbooks by segment

## Core Practices

- Define onboarding success criteria with the customer
- Map milestones from kickoff to first value to steady state
- Assign owners (CS, PS, customer stakeholders)
- Track leading indicators of adoption during onboarding
- Escalate blockers early (technical, political, resource)
- Capture learnings into product and sales feedback loops

## Principles

- Speed to value beats exhaustive training theater
- Mutual action plans work post-sale as well as pre-sale
- Over-promising in sales creates onboarding debt
- Onboarding ends when value is realized, not when a checklist is emailed

## Verification

- [ ] Success criteria are defined per segment/product
- [ ] Milestone progress is visible
- [ ] Early risk triggers exist and are acted on
