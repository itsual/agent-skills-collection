# Total Rewards

## Overview

Total rewards is the full package employees receive for their work: tangible and intangible. Strategy connects the pieces so the offer is competitive and coherent.

## When to Use

- Designing employee value proposition (EVP)
- Competing for talent beyond base pay
- Communicating rewards holistically
- Balancing cost with perceived value

## Core Practices

- Map all reward elements employees actually value
- Align pay, benefits, recognition, and career growth deliberately
- Segment rewards where workforce needs differ materially
- Communicate total value clearly at offer and ongoing
- Measure perceived value, not only program cost
- Govern trade-offs when budgets tighten

## Principles

- Unintegrated programs feel like random perks
- Perception gaps waste spend
- Career growth is often a top “reward” for ambitious talent
- Transparency about how rewards work builds trust

## Verification

- [ ] EVP/rewards elements are explicitly defined
- [ ] Offer communication includes total picture
- [ ] Feedback on perceived value is collected
