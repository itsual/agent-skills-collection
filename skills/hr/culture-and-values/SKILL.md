# Culture and Values

## Overview

Culture is the pattern of behaviors that are rewarded, punished, and tolerated. Values statements matter only when systems and leaders reinforce them.

## When to Use

- Defining or refreshing values
- Culture assessment and change
- Aligning hiring and performance to values
- Post-merger culture integration

## Core Practices

- Describe values as observable behaviors
- Embed values in hiring, recognition, and performance criteria
- Surface hypocrisy gaps between stated and lived culture
- Use leader storytelling and modeling deliberately
- Measure culture through employee signals and behavioral indicators
- Change systems (incentives, rituals, org design) not only messaging

## Principles

- What leaders tolerate becomes culture
- Values that conflict with incentives lose
- Subcultures exist — diagnose before prescribing one narrative
- Culture change is slow and evidence-based, or it is slogans

## Verification

- [ ] Values have behavioral definitions
- [ ] People processes reference them concretely
- [ ] Gaps between stated and lived culture are acknowledged
