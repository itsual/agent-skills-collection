# Statistical Analysis

## Overview

Statistical analysis quantifies patterns and uncertainty. Good practice matches methods to data structure and communicates limits of inference.

## When to Use

- Summarizing business or research metrics
- Estimating effects with uncertainty
- Comparing groups responsibly
- Supporting decisions with quantified evidence

## Core Practices

- Choose statistics that match the scale and distribution of data
- Report uncertainty (intervals, standard errors) not only point estimates
- Check assumptions for the methods used
- Distinguish description from causal claims
- Use effect sizes alongside significance where relevant
- Visualize distributions, not only summary tables

## Principles

- Precision theater without uncertainty misleads
- Statistical significance ≠ practical importance
- Sample bias cannot be fixed by fancier math alone
- Simpler methods well-applied beat complex methods misapplied

## Verification

- [ ] Methods fit the data and question
- [ ] Uncertainty is communicated
- [ ] Claims match the strength of evidence
