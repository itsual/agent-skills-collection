# Storytelling with Data

## Overview

Analysis creates value only when understood and acted on. Storytelling structures findings so busy decision-makers grasp the point and the stakes.

## When to Use

- Executive presentations of analysis
- Insight write-ups and readouts
- Turning complex results into choices
- Training analysts on communication

## Core Practices

- Lead with the answer, then support with evidence
- Use a clear arc: context → finding → implication → action
- Choose one primary message per slide/section
- Show the minimum chart needed to prove the point
- Anticipate objections and address uncertainty honestly
- End with specific recommended next steps

## Principles

- Your notebook is not the deliverable
- Audience altitude determines detail depth
- Surprise claims need stronger visible evidence
- Recommendations should follow from findings, not precede them

## Verification

- [ ] A listener can restate the main insight
- [ ] Evidence is visible and sufficient
- [ ] Actions are concrete and owned
