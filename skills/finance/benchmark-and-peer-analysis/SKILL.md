# Benchmark and Peer Analysis

## Overview

Benchmarks answer “compared to what?” Peer and internal benchmarks ground targets and highlight structural gaps — when definitions are comparable.

## When to Use

- Target setting and planning
- Investor and board context slides
- Diagnosing margin or efficiency gaps
- Competitive financial positioning

## Core Practices

- Choose relevant peer sets (stage, model, geography)
- Normalize definitions before comparing
- Use multiple sources; treat any single benchmark as directional
- Combine external benchmarks with internal historical trends
- Focus on actionable gaps, not vanity ranking

## Principles

- Bad peer sets produce false comfort or false panic
- Stage-adjusted comparisons matter for startups
- Outliers need investigation, not only celebration
- Benchmarks inform judgment; they don’t replace strategy

## Verification

- [ ] Peer set rationale is stated
- [ ] Metric definitions are reasonably comparable
- [ ] Gaps link to potential actions
