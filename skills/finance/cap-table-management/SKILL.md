# Cap Table Management

## Overview

The cap table records who owns what. Errors create legal, fundraising, and employee-trust problems. Scenario modeling informs dilution decisions.

## When to Use

- Equity and option grants
- Fundraising dilution modeling
- Convertible note / SAFE conversions
- Due diligence preparation

## Core Practices

- Keep a single source of truth (software or controlled spreadsheet)
- Record all instruments: common, preferred, options, warrants, convertibles
- Model fully diluted ownership correctly
- Track vesting, exercises, and cancellations
- Align legal docs with the cap table

## Principles

- Ambiguity on ownership is existential
- Model rounds before term sheets are signed
- Employee option pool decisions affect culture and hiring
- Cap table hygiene is a continuous process

## Verification

- [ ] Cap table matches legal records
- [ ] Fully diluted view is available and understood
- [ ] Round scenarios are modeled before negotiation ends
