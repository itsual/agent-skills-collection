# Financial Storytelling

## Overview

Numbers without narrative don’t drive action. Financial storytelling selects what matters, explains why, and points to implications — without spin.

## When to Use

- Management commentary and MBRs
- Board and investor presentations
- Cross-functional performance reviews
- Training FP&A on executive communication

## Core Practices

- Lead with the headline, then support with detail
- Use bridges and waterfalls for movement
- Contrast actual vs expectations explicitly
- Separate facts, interpretations, and recommendations
- Match depth to audience

## Principles

- Clarity is respect for the audience’s time
- Honesty about uncertainty builds long-term credibility
- Jargon is not sophistication
- Every story should answer “so what?”

## Verification

- [ ] Audience can state the main message after one pass
- [ ] Material movements are explained with drivers
- [ ] Recommendations follow from the analysis
