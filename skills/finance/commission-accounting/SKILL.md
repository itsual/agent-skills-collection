# Commission Accounting

## Overview

Commissions are both a sales motivator and an accounting complexity. Accruals, true-ups, and capitalization rules must match plan reality.

## When to Use

- Monthly commission accruals
- Comp plan changes
- Commission system implementations
- Audit of commission expense

## Core Practices

- Accrue based on earned commissions, not only paid
- Reconcile CRM/closed-won to commission calculations
- Understand when contract cost capitalization applies
- True-up promptly when plans or deals change
- Keep plan documentation synchronized with finance logic

## Principles

- Shadow spreadsheets for comp create audit and trust risk
- Under-accrual creates nasty surprises
- Plan complexity multiplies accounting complexity
- Sales ops and finance need one calculation truth

## Verification

- [ ] Accruals match plan earning rules
- [ ] Payouts reconcile to approved calculations
- [ ] Documentation supports audit inquiries
